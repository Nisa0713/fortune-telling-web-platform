# Fortune Telling Web Platform

An interactive fortune telling website that allows users to explore horoscope and tarot card readings.

## Features

- User registration and login
- Horoscope readings
- Tarot card selection system
- Interactive card drawing interface

## Technologies

- PHP
- HTML
- CSS
- JavaScript
- MySQL

## Database

The project includes a database setup file: `database.sql`
