# Fortune Telling Web Platform

An interactive fortune telling website that allows users to explore horoscope and tarot card readings.

## Features

- User registration and login
- Horoscope readings
- Tarot card selection system
- Interactive card drawing interface

## Technologies

- PHP
- HTML
- CSS
- JavaScript
- MySQL

## Database

The project includes a database setup file: `database.sql`

## Live Demo

The website was previously deployed on a hosting server.
Due to temporary hosting and security configuration updates, the live demo is currently unavailable.

The project source code and database structure are fully available in this repository.
The live demo will be restored soon.

## How to Run Locally

1. Install **XAMPP** or any PHP server.
2. Import the `database.sql` file into MySQL.
3. Place the project files inside the `htdocs` directory.
4. Start Apache and MySQL.
5. Open in browser:  http://localhost/astro-project/
